# என் பெயர் • En Peyar

> AI-powered Tamil-first startup, project, and brand name generator.

Generate meaningful startup, project, app, and brand names using Tamil, English, and Tanglish.

Built for founders, developers, students, creators, and hackathon teams.

---

## ✨ Why En Peyar?

Most name generators produce random combinations that feel generic.

En Peyar focuses on meaningful names inspired by:

* Tamil
* English
* Tanglish
* Startup culture
* Modern branding principles

The goal is simple:

Help creators find names that are memorable, meaningful, and launch-ready.
